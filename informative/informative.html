Anime Subtitle Study - Information

Welcome to Anime Subtitle Study! Here is a guide to help you understand and use all the features of this site.

Features

  • Upload Video/Audio: Click the "Upload Video/Audio" button to upload your media files. Supported formats include .mp4, .mkv, .avi, .mov, .wmv, .flv, .webm. The video will be played in the player for subtitle synchronization.
  • View Full Subtitles: Click the "View Full Subtitles" button to see all the subtitles loaded for the video in a separate modal window. This helps you review the entire subtitle file at once.
  • Pause on Unknown Words: Enable or disable automatic pausing when unknown words appear in the subtitles by clicking the "Enable Pause" or "Disable Pause" buttons. When enabled, the video pauses for 3 seconds whenever an unknown word appears, giving you time to read and understand it.
  • Fullscreen Mode: Click the "Fullscreen" button to toggle fullscreen mode for the video player. This is useful for an immersive viewing experience.
  • Retiming Buttons: Show or hide retiming buttons by clicking the "Show Retiming Buttons" button. These buttons allow you to adjust the timing of subtitles to match the video more accurately.
  • Export/Import Unknown Words: Save or load your list of unknown words using the "Export Unknown Words" and "Import Unknown Words" buttons. Exporting creates a JSON file of all unknown words, and importing allows you to load a previously saved list.
  • Toggle Subtitle Panel: Use the arrow buttons to show or hide the subtitle panel for a cinema-like viewing experience. This hides the subtitle panel, centering the video player.

Keyboard Shortcuts

  • S: Capture a frame from the video.
  • Left Arrow: Navigate to the previous subtitle.
  • Right Arrow: Navigate to the next subtitle.
  • Up Arrow: Navigate to the previous subtitle.
  • Down Arrow: Navigate to the next subtitle.

Unknown Words

When an unknown word appears in the subtitles, it will be highlighted in yellow. You can edit the unknown words list by clicking the "Edit Unknown Words" button next to each subtitle.

Importing Unknown Words

Using the Migaku Legacy extension, you can export all unknown words to Anki as cards in a separate deck called "unknown words". When creating this deck, make sure to create a specific card type that only contains the front and back fields. On the front, add the unknown word, and on the back, add its definition. This ensures that Migaku only creates cards for the words and their meanings. Then, export these cards within Anki as a .txt file.

Method 1: Using the Conversion Script

  1. Go to the main page and click on the "Convert Unknown Words" button.
  2. Upload the .txt file exported from Anki.
  3. Click the "Convert" button to format the file.
  4. Download the converted .txt file.
  5. Upload the converted .txt file to the main page to import the unknown words.

Method 2: Using ChatGPT

If you need to translate the definitions into your native language, you can use ChatGPT to assist you. Use the following prompt with ChatGPT:

                
From now on, I will send you photos or txt files of Japanese subtitles.
Your role is to identify the unknown words (when they are marked or have
a red underline under them. Usually, the furigana reading appears on these
words) and create a list with all these words plus their meanings in
(your native language) according to the sentence. You should create the
list in this format:

調べ上げ (しらべあげ) = investigate
息の根 (いきのね) = vitality

And so on. Create precise translations that convey the actual meaning of
the word. If you understand, I will send you the first words. Do not
translate proper names. Translate ALL words underlined in red or highlighted.
If it is a TXT file, just take the Japanese words present and create a list
with the same format indicated above. Do not repeat words. Create a unique
list for me to copy. Make sure not to add the kanji form as the word. For
example, in the txt ガス焜炉 is in its katakana form ガスコンロ. The word you
translate should always be at the beginning of the information. For example,
in the txt it says "ガスコンロ がすこんろ【ガス焜炉】 gas stove, gas cooker, gas range"
so ガスコンロ should be on the list as the unknown word and not its kanji form.

Copy the list generated by ChatGPT and paste it into a new .txt file. Then upload this .txt file to the site, and it will automatically recognize the words in the subtitles and add the unknown words to the list.
